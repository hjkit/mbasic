MISSING FEATURES FROM MBASIC 5.21 SPEC:

=== HIGH PRIORITY (commonly used) ===

1. PRINT USING - Formatted output with format strings
   Status: Not implemented
   Impact: Cannot format numbers/strings for output

2. PRINT# USING - Formatted file output
   Status: Not implemented
   Impact: Cannot format file output

3. TAB(n) - Tab to column n in PRINT statements
   Status: Not implemented
   Impact: Cannot tab to specific columns

4. SPC(n) - Print n spaces in PRINT statements
   Status: Not implemented
   Impact: Cannot insert spaces easily

=== MEDIUM PRIORITY (useful but has workarounds) ===

5. SWAP var1, var2 - Exchange values
   Status: Parsed but not executed
   Impact: Minor convenience

6. NULL n - Set number of nulls after CR
   Status: Not implemented
   Impact: Only needed for slow terminals

7. TRON/TROFF - Trace execution
   Status: Not implemented
   Impact: Debugging aid, not critical

=== LOW PRIORITY (specialized features) ===

8. CVI/CVS/CVD - Convert string to integer/single/double
   Status: Not implemented
   Impact: Needed for reading binary data from random files

9. MKI$/MKS$/MKD$ - Convert integer/single/double to string
   Status: Not implemented
   Impact: Needed for writing binary data to random files

10. VARPTR(var) - Get variable address
    Status: Not implemented
    Impact: Only for advanced assembly language interface

11. FRE(n) - Free memory
    Status: Not implemented
    Impact: Memory management, less critical in modern systems

12. LPOS(n) - Printer position
    Status: Not implemented
    Impact: Printer support not needed

13. LPRINT/LPRINT USING - Print to printer
    Status: Not implemented
    Impact: Printer support not needed

14. LLIST - List to printer
    Status: Not implemented
    Impact: Printer support not needed

=== NOT NEEDED (hardware/system specific) ===

15. CLOAD/CSAVE - Cassette tape operations
    Status: Not implemented
    Impact: Obsolete hardware

16. WIDTH - Set output width
    Status: Parsed but not executed
    Impact: Terminal control, not critical
