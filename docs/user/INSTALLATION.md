# MBASIC Installation Guide

**Status:** PLACEHOLDER - Documentation in progress

## Quick Install

### For Users (pip)
```bash
pip install mbasic-interpreter
```

### For Developers (from source)
```bash
git clone https://github.com/avwohl/mbasic.git
cd mbasic
pip install -r requirements.txt
python3 mbasic.py
```

## Requirements

- Python 3.8 or later
- Optional: urwid (for Curses UI)
- Optional: nicegui (for Web UI)
