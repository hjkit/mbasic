## Quick Reference

### Essential Commands

- **F2** or **Ctrl+R** - Run program
- **F5** or **Ctrl+S** - Save program
- **F9** or **Ctrl+O** - Load program
- **F1** - Open help
- **ESC** - Clear errors
- **Q** - Quit

### Navigation

- **Up/Down** - Move between lines
- **Left/Right** - Move cursor
- **Home/End** or **Ctrl+A/E** - Start/end of line
