# Using the Help System

How to navigate MBASIC help.

## Opening Help

Press **F1** anytime to open help.

**Note:** If you don't have function keys, you can access help by looking in the `docs/help/` directory.

## Navigation Keys

### Scrolling

| Key | Action |
|-----|--------|
| **Up/Down** | Scroll one line |
| **Space** | Page down (full screen) |
| **B** | Page up |

### Following Links

Links appear in brackets like [this](link).

1. Use **Up/Down** to move cursor to a link
2. Press **Enter** to follow the link

### Going Back

| Key | Action |
|-----|--------|
| **U** | Go up to parent topic or back in history |

### Browsing Topics

| Key | Action |
|-----|--------|
| **N** | Next link in current page |
| **P** | Previous link in current page |

### Exiting Help

| Key | Action |
|-----|--------|
| **Q** or **ESC** | Exit help, return to editor |
