# Keyboard Shortcuts

## Editor

- **^R** - Run program
- **^P** - Show help
- **^Q** - Quit IDE
- **^C** - Cancel/Close dialogs
- **ESC** - Cancel/Close dialogs
- **^X** - Open menu

## Debugging

- **b** - Toggle breakpoint on current line (shows ● indicator)
- **Mouse click** - Click on the ● or space at the start of a line to toggle breakpoint
- When breakpoint hits (status line appears at top):
  - **c** - Continue to next breakpoint or end
  - **s** - Step through line by line
  - **e** - End execution and return to editor

## Menu Navigation

- **↑/↓** or **Tab** - Navigate menu items
- **Enter** - Select menu item
- **ESC** - Close menu

## Help Browser

- **↑/↓** - Scroll line by line
- **Space** - Page down
- **B** - Page up
- **Enter** - Follow link at cursor
- **U** - Go back/up to parent
- **N** - Next topic
- **P** - Previous topic
- **Q** or **ESC** - Exit help
