## Navigation

- **Up/Down** - Scroll through help
- **Space** - Page down
- **Enter** - Follow a link
- **U** - Go up to parent topic
- **Q** or **ESC** - Exit help

## Tips

- Press **F1** or **H** to open help
- Cursor on BASIC keyword + **F1** = context help
- UI commands are in UI-specific sections
- Language features are in Language Reference
