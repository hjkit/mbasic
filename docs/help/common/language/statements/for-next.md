# FOR ••• NEXT

## Syntax

```basic
FOR <variable>=x TO y [STEP z]
NEXT [<variable>] [,<variable> ••• ]
where x, y and z are numeric expressions.
```

**Versions:** SK, Extended, Disk

## Purpose

To allow a series of instructions        to         be performed in a loop a given number of times.
