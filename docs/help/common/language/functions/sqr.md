# SQR

## Syntax

```basic
SQR(X)
```

## Description

Returns the square root of X.   X must be >=0.

## Example

```basic
10 FOR X = 10 TO 25 STEP 5
             20 PRINT X, SQR(X)
             30 NEXT
             RUN
              10            3.16228
              15            3.87298
              20            4.47214
              25            5
             Ok
```
