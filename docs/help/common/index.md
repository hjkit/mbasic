# MBASIC IDE Help

Welcome to the MBASIC Interpreter IDE!

## Quick Start

- **^R** - Run the current program
- **^P** - Show this help
- **^Q** - Quit the IDE
- **^X** - Open menu

## Navigation

- **↑/↓** - Scroll up/down
- **Space** - Page down
- **B** - Page up
- **Enter** - Follow link
- **U** - Go back/up
- **Q** or **ESC** - Exit help

## About

This is a MBASIC-80 compatible interpreter with a modern IDE.
It supports classic BASIC programs from CP/M and early microcomputers.
