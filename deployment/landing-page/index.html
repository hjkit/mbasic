What is MBASIC?

MBASIC is a faithful recreation of Microsoft BASIC-80 version 5.21, the classic BASIC interpreter that ran on CP/M systems in the late 1970s and early 1980s. This implementation brings the vintage computing experience to your modern browser.

Features

  • Full MBASIC 5.21 language compatibility
  • Web-based IDE with syntax highlighting
  • Line-by-line debugging and breakpoints
  • Variable inspection and modification
  • Multiple UI backends (Web, Terminal, GUI)
  • Compiler to CP/M .COM executables
  • Extensive documentation and examples
  • Open source (MIT License)

Quick Start

10 PRINT "HELLO, WORLD!" 20 FOR I = 1 TO 10 30 PRINT "COUNT: "; I 40 NEXT I 50 END

Click "Launch Web IDE" above to start programming in MBASIC right away. No installation needed!

About

MBASIC is developed and maintained by the vintage computing community. It aims to preserve and celebrate the heritage of early microcomputer programming by making these classic systems accessible to modern users and new generations of programmers.

Special thanks to all contributors and the vintage computing community for their support and feedback.
